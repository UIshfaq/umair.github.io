SaaS Mariage — Invitations Numériques

Vue.js 3 FastAPI PostgreSQL

Le Concept

"Offrir une interface ultra-simple pour des utilisateurs non-techniques tout en garantissant une robustesse technique en backend pour la gestion de milliers d'invitations."

L'Enjeu Technique

Le projet nécessitait une architecture capable de gérer des données hautement relationnelles (familles, groupes, tables) avec une interface réactive permettant la personnalisation visuelle en temps réel.

Scalabilité & Typage Fort

Features Avancées

Builder Dynamique

Personnalisation en temps réel des invitations avec Vue.js 3. Gestion granulaire des styles, polices et imports médias.

Gestion RSVP

Suivi des réponses automatisé, gestion des allergies alimentaires et segmentation par groupes d'invités.

Plan de Table

Outil drag-and-drop pour l'organisation des salles, optimisé pour une utilisation fluide sur tablette et desktop.

Stockage S3

Gestion des médias via AWS S3-compatible pour un stockage persistant et sécurisé des souvenirs numériques.

Architecture Backend

Utilisation de FastAPI pour une gestion asynchrone performante. Base de données PostgreSQL optimisée pour les requêtes relationnelles complexes. Code structuré en modules extensibles et validation de données via Pydantic.
